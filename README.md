# Conversor de divisas

Este proyecto es una aplicación web que permite convertir montos entre diferentes monedas en tiempo real utilizando datos actualizados de una API externa.

## Características

- Convierte montos entre varias divisas utilizando tasas de cambio actualizadas.
- Una interfaz de usuario sencilla y fácil de usar.
- Admite una amplia variedad de monedas internacionales.

## Instrucciones de Uso

1. Use las entradas para el monto que desea convetir.
2. Selecciona la moneda de origen y la moneda de destino.
3. Haz clic en el botón de conversión para ver el resultado.

## Tecnologías Utilizadas

- React y css para la estructura, diseño y funcionalidad del conversor de divisas.
- Api para la conversion de divisas **Currency beacon**
- Api para la obtencion de distintas monedas **Fixer**
